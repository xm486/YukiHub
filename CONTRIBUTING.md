# Contributing to YukiHub

感谢你对 YukiHub 的关注。

## 如何提交问题

提交 Issue 时，建议提供：

- 设备型号
- Android 版本
- YukiHub 版本
- 问题复现步骤
- 截图或录屏
- 相关日志

## 提交代码

如果你想提交 Pull Request，建议：

1. 先 Fork 仓库
2. 新建分支
3. 完成修改并测试
4. 提交清晰的 commit message
5. 发起 Pull Request

## 建议的修改范围

- UI 优化
- 功能修复
- 同步逻辑改进
- 兼容性优化
- 文案与文档完善

## 约定

- 请尽量保持代码风格一致
- 修改功能时尽量附带说明
- 如涉及权限、同步、数据迁移，请先说明影响范围

## 开源协议

本项目采用 GPL-3.0 协议。
参与贡献即表示你接受以 GPL-3.0 方式发布你的贡献内容。
